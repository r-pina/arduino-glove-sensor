# arduino-glove-sensor
Arduino obstacle detection system using ultrasonic sensors and buzzer

## Features
- Detects objects in front, left, and right directions
- Activates a buzzer when obstacles are detected
- Written in C++ using the Arduino framework

## Hardware
- Arduino board
- 3 Ultrasonic sensors
- Buzzer

I programmed the system in C++ and implemented the sensor detection logic.
